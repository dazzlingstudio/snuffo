Uvendia is the Asset Management System that runs on Umbraco 8. Use Uvendia to manage Products and Events that you want to sell on your e-commerce website. 
This component also provides you the functionalities to send transactional order e-mails, generate PDF invoices and e-tickets for your customers without additional programming. 
For a full demo take a look at demo.snuffo.com.

Quick Start
=> Make sure you run the database script located at '\umbraco\uvendia\content\uvendia.sql'. You need this script in order to start using Uvendia. 
=> Don't forget to build the project before start using Uvendia!
=> For a quick start visit our website www.uvendia.com and go to documentation > tutorial.

Assign Access to Uvendia
At first you don't have access to Uvendia section within Umbraco. Please visit www.uvendia.com and go to documentation > tutorial to understand how you can get access to the Uvendia section.
